Key Features

Transformer Model

State-of-the-art deep learning architecture with self-attention mechanism for accurate predictions.

Attention Visualization

Visualize which amino acids the model focuses on when making predictions with interactive heatmaps.

Feature Analysis

Detailed analysis of amino acid composition and feature importance scores with interactive charts.

Fast Prediction

Real-time prediction with high accuracy and confidence scores for both single and batch sequences.

Export Results

Download prediction results in multiple formats including CSV, JSON, and detailed PDF reports.

Responsive Design

Fully responsive interface that works seamlessly on desktop, tablet, and mobile devices.

Model Information

Model Architecture
  • Architecture Transformer Encoder
  • Layers 2
  • Attention Heads 8
  • Embedding Dimension 256
  • Hidden Dimension 384
  • Dropout Rate 0.3
Training Details
  • Dataset Size 423 Sequences
  • Class Balance 50/50
  • Validation Split 80/20
  • Optimizer AdamW
  • Loss Function Binary Cross Entropy
  • Training Epochs 100
About AntiCancerPeptide Predictor

This web application uses a Transformer-based deep learning model to predict anticancer peptides. The model was trained on a curated dataset of known anticancer and non-anticancer peptides.
